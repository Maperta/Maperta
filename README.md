# Maperta

Maperta is a 3D map website that shows the landmarks and historical events of Shenzhen, China. It is created by a group of high school students, and you can interact with the 3D models by clicking on them to see the information about that certain landmark or area.

The website is still under construction, and the estimated release date is May 20th. You can find our source code above. Feel free to use it and make some changes.
